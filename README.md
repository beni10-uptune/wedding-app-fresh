# UpTune for Weddings

Transform wedding music planning from stress to joy. The collaborative platform that helps couples, guests, and DJs create the perfect wedding soundtrack.

## 🎵 About

UpTune for Weddings is a specialized platform designed to solve the biggest pain point in wedding planning: music selection. We transform this stressful process into the most enjoyable part of wedding planning through:

- **Joyful Collaboration**: Build your soundtrack together with your partner and guests
- **Expert Curation**: Start with 500+ proven wedding hits, not a blank page
- **Effortless Organization**: Export perfectly organized playlists for your DJ

## 🚀 Features

### For Couples
- **Curated Music Library**: 500+ wedding songs organized by moment and mood
- **Partner Collaboration**: Work together to build your perfect playlist
- **Guest Involvement**: Let guests suggest and vote on songs
- **Multiple Playlists**: Separate playlists for ceremony, cocktails, dinner, and dancing

### For DJs
- **Professional Export**: One-click export with all necessary details
- **Organized Requests**: No more messy Google Docs or scattered lists
- **Timing Notes**: Special requests and timing information included
- **Multiple Formats**: Export to various playlist formats

### For Guests
- **Easy Suggestions**: Simple interface to suggest favorite songs
- **Voting System**: Vote on suggested songs to help couples decide
- **Real-time Updates**: See the playlist evolve in real-time

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/your-username/weddings.uptune.xyz.git
cd weddings.uptune.xyz
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production
```bash
npm run build
npm start
```
